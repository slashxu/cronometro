## 🚀 Instalando Projeto

Para instalar o projeto, siga estas etapas:

Clone o repositório
```
https://github.com/slashxu/cronometro.git
```

Baixe as dependências
```
npm install ou yarn install
```

Rode o app
```
npx react-native run-android
```
